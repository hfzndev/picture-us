# Picture Us App

This is the Next.js application directory for Picture Us.

For full project documentation, including the tech stack, architecture, and developer guides, please see the [root README](../README.md) and the [docs/ directory](../docs/).

## Quick Commands
- `pnpm dev`: Start development server
- `pnpm build`: Build for production
- `pnpm start`: Start production server
- `pnpm lint`: Run ESLint
